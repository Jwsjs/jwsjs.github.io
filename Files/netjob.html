Full Stack .NET Developer

Remote
£550-£650 per day
.NET Core, ASP.NET, C#, SQL Server, React:

We are seeking a skilled and experienced Full Stack .NET Developer with expertise in ASP.NET, C#, SQL Server, and front-end frameworks such as Angular or React to join our dynamic development team. As a Full Stack .NET Developer, you will be responsible for designing, developing, and maintaining robust and scalable web applications that deliver exceptional user experiences. Your expertise in both back-end and front-end technologies will be crucial in building end-to-end solutions.



Responsibilities:

  • Collaborate with cross-functional teams, including designers and other developers, to translate business requirements into robust and scalable web applications using ASP.NET and C#.
  • Develop and maintain database schemas, stored procedures, and SQL queries for efficient data storage and retrieval using SQL Server.
  • Implement responsive designs and ensure compatibility across multiple platforms and devices using Angular or React.
  • Create RESTful APIs and handle data exchange between the front-end and back-end components.
  • Optimize application performance by employing best practices, code optimization techniques, and performance testing.
  • Troubleshoot and debug application issues, and provide timely resolutions.
  • Stay up-to-date with the latest industry trends, frameworks, and tools related to .NET development and front-end technologies.
  • Collaborate with the QA team to ensure high-quality software deliverables through code reviews, automated testing, and continuous integration processes.
  • Participate in agile development processes, including sprint planning, daily stand-ups, and retrospective meetings.
  • Mentor and provide technical guidance to junior developers, fostering a collaborative and knowledge-sharing environment.

    • Requirements:

  • Bachelor's degree in Computer Science, Software Engineering, or a related field (or equivalent work experience).
  • Proven experience as a Full Stack .NET Developer, with a strong portfolio demonstrating expertise in ASP.NET, C#, and SQL Server.
  • Proficiency in front-end technologies such as Angular or React for building responsive user interfaces.
  • Experience with database design and development using SQL Server or similar relational database systems.
  • Familiarity with RESTful API development and integration.
  • Strong understanding of software development principles, SOLID principles, and design patterns.
  • Familiarity with version control systems (e.g., Git) and collaborative development workflows (e.g., Agile, Scrum).
  • Knowledge of performance optimization techniques and tools for web applications.
  • Excellent problem-solving skills and attention to detail.
  • Strong communication and collaboration skills, with the ability to work effectively in a team environment.
